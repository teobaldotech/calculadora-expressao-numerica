# 📐 Resolvedor de Expressão Numérica

Um pequeno aplicativo em HTML, CSS e JavaScript que permite calcular expressões matemáticas diretamente no navegador.  
Suporta operações básicas (`+`, `-`, `*`, `/`), parênteses e potência (`^`).

---

## 🚀 Funcionalidades
- Entrada de expressões matemáticas em um campo de texto.
- Conversão automática de `^` para `**` (potência).
- Validação de caracteres permitidos para maior segurança.
- Exibição do resultado ou mensagem de erro.
- Interface simples e responsiva.

---

## 🖥️ Como usar
1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Digite uma expressão matemática no campo de entrada.  
   Exemplo:  (5+3)*2^3-10/2

   ⚙️ Tecnologias utilizadas
- **HTML5** para estrutura da página
- **CSS3** para estilização
- **JavaScript** para lógica de cálculo
